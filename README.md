Всех приветствую на моей первой проектной работе! 
Идея была создать корпоративный интернет-магазин, для сотрудников парков и скверов, где будет своя внутренняя валюта, которую можно получить путем выполнения, некоторых типов задач, с проверкой от админа.

Писался весь back на Python 3.13, с использованием Django 5.2.8. База данных - PostgreSQL. Frontend часть написана - HTML, Boostrap, JS.

На данный момент, большая часть функционала выполнена, а именно:
1. Регистраци\Вход
2. Зарегистрировавшийся сотрудник, имеет доступ к оформлению товара, предложенным магазином
3. Можно выбирать товар, изменять количество
4. Полностью функционирует этапы покупки (товар в корзине, товар оформлен, после можно посмотреть историю заказов в личном кабинете)
5. Реализована часть оформления (для кого, количество, удаление товаров)
6. Просмотр личных данных
7. Общение с админом
8. Ответ сотрудником производится через админку django

В будущем нужно будет реализовать:
1. Прикрепления фотографий
2. Подключить Redis, Celery
3. Завернуть до конца все по контейнерам
4. Написать тесты
5. Продать этот проект ребятам из Парков и Скверов :D
